# 如何成为 [janusgraph.cn](https://github.com/simon824/janusgraph.cn) 社区的 contributor

## 1. 首先点击右上角 fork [janusgraph.cn](https://github.com/simon824/janusgraph.cn) 项目到自己的仓库

![fork](https://github.com/simon824/janusgraph.cn/blob/main/images/fork.jpg)

## 2. 在自己仓库中编辑内容

![fork](https://github.com/simon824/janusgraph.cn/blob/main/images/rep.jpg)

## 3. 发起 pull requests 

![fork](https://github.com/simon824/janusgraph.cn/blob/main/images/pr.jpg)

## 4. 提交成功，等待项目成员 review，保证内容质量

![fork](https://github.com/simon824/janusgraph.cn/blob/main/images/prs.jpg)

## 5. review 完成，pr 即会被 merge 到主分支。

![fork](https://github.com/simon824/janusgraph.cn/blob/main/images/merge.jpg)

至此，恭喜你成为 [janusgraph.cn](https://github.com/simon824/janusgraph.cn) 社区的 contributor 成员 ！
